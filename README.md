# HEALTH ARCHI スマート見積システム

木造工務店の新築見積を、誰でも5分で出せる無料ツール。

## 公開URL

- 本体：https://[アカウント名].github.io/healtharchi-smart-quote/
- 使い方ガイド：https://[アカウント名].github.io/healtharchi-smart-quote/guide.html

## 特徴

- 136項目の標準単価マスタ付き（自社単価に書き換えて永久使用）
- 坪数・グレード・建物形状を入れるだけで自動見積生成
- 物件ごとに複数見積を複製可能（第1回／第2回など）
- 印刷／PDF保存対応（シンプル／内訳付き／正式の3書式）
- データはブラウザ内に保存（当社サーバーには一切送信されません）
